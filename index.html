NEM Developer Documentation¶

Welcome to the NEM Developer documentation, the place where you’ll find everything you need to start integrating NEM blockchain into your application.

Note

This documentation refers to NEM latest version, Catapult, which provides higher performance and new functionalities to NEM.

If it is your first time developing with blockchain technology, this guide is for you. NEM was created to be your entry point into the blockchain industry. If you’re an experienced developer, you will learn how to create cutting-edge blockchain apps.
